# SwagLabs Selenium Automation Project

## Overview
This project contains a Selenium WebDriver automation framework for the SwagLabs demo website.  
It demonstrates automated testing of key user scenarios using Java, TestNG, and Page Object Model (POM).

---

## Test Scenarios
1. **Login with an Invalid User** – Verify that a locked-out user cannot log in.  
2. **Login with a Valid User** – Verify successful login and redirection to the inventory page.  
3. **Filter Products by Name and Price** – Verify correct product sorting.  
4. **Add Items to Cart** – Verify adding multiple items.  
5. **Perform Checkout** – Verify order completion and confirmation.

---

## Technologies Used
- Java 17  
- Selenium WebDriver 4.34.0 
- TestNG 7.11.0  
- WebDriverManager 5.7.0  
- Maven  
- Page Object Model (POM) design pattern
